# vue-skeleton-layout

skeleton layout grid system for VueJS.

## Installation

`npm install vue-skeleton-layout` or `yarn add vue-skeleton-layout`

### ES6
```js

import VueSkeletonLayout from 'vue-skeleton-layout'

export default {
  ...
  components: {
    VueSkeletonLayout
  },
  ...
}
```
After that, you can use it in your templates:

```template
<vue-skeleton-layout />
```
